# Maiskolben

Code für die Lötstation "Maiskolben"
mit ein paar Anpassungen für das Display (ST7735-Chip)

## Display Pin-Zuordnung
|Pin Display|Benennung|Pin Arduino|
|-----------|---------|-----------|
|1          |GND      |GND        |
|2          |VCC      |5V         |
|3          |SCK      |D13        |
|4          |SDA      |D11        |
|5          |RES      |A1         |
|6          |RS       |A5         |
|7          |CS       |D10        |
|8          |LEDA     |3V3        |
